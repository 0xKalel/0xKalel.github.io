# Blog Post Template

Use this template when creating new blog posts. Copy the structure and fill in your content.

---

## File Naming

**Format**: `_posts/YYYY-MM-DD-descriptive-title-slug.md`

**Example**: `_posts/2025-11-02-chrome-devtools-mcp-wsl-claude-code.md`

---

## Front Matter Template


## Quick Checklist Before Publishing

Before committing your post, verify:

- [ ] Title is 50-60 characters with primary keyword
- [ ] Description is 150-160 characters, compelling
- [ ] 3-5 categories selected
- [ ] 5-10 relevant tags added
- [ ] Primary keyword in first 100 words
- [ ] H2 headings are keyword-rich and descriptive
- [ ] 2-5 internal links to other posts/pages
- [ ] 3-8 external authoritative links
- [ ] Code blocks have language specified
- [ ] Conclusion includes CTA with links
- [ ] All images have descriptive alt text (if applicable)
- [ ] Post is 1500+ words (for comprehensive guides)
- [ ] Readability: short paragraphs, lists, clear structure
