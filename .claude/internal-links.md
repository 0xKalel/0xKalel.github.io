## Internal Linking Strategy

### Minimum Requirements
Every blog post should have:
- **2-5 internal links** to other posts or pages
- **1 link to About page** (when relevant)
- **1 link to Blog index** (in conclusion)

### Best Practices
1. Use descriptive, keyword-rich anchor text
2. Link naturally within content context
3. Don't force links where they don't fit
4. Vary anchor text for the same destination
5. Update older posts to link to new posts when relevant

### Example Good Internal Links

```markdown
<!-- Linking to Chrome DevTools MCP post -->
If you're working with browser automation, check out my [comprehensive guide to installing Chrome DevTools MCP in WSL for Claude Code](/blog/2025/11/02/chrome-devtools-mcp-wsl-claude-code/).

<!-- Alternative anchor text -->
I covered this in detail in my [Chrome DevTools MCP setup tutorial](/blog/2025/11/02/chrome-devtools-mcp-wsl-claude-code/).

<!-- Linking to About page -->
As I mentioned in [my background](/about/), I work extensively with AI-assisted development workflows.

<!-- Linking to Blog -->
For more tutorials like this, explore [all my blog posts](/blog/).
```
